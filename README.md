# Proyecto SQL - El Empleo

Este proyecto consiste en el diseno de una base de datos inspirada en la estructura y funcionamiento de la pagina **El Empleo**.

## Descripcion del proyecto

Para desarrollar este trabajo, primero se observo como estaba compuesta la pagina, identificando su estructura, la informacion que maneja y las necesidades principales del sistema. Con base en esa obtencion de requisitos, se realizo el diseño de las tablas necesarias para representar la informacion de forma organizada dentro de la base de datos.

Ademas de las tablas, tambien se elaboraron vistas y funciones SQL con el fin de facilitar la consulta, el analisis y la manipulacion de los datos segun los requerimientos identificados.

## Contenido del proyecto

- `El empleo1.sql`: contiene la estructura principal de la base de datos y la creacion de las tablas.
- `7.VistasYFunciones.sql`: contiene las vistas y funciones desarrolladas para complementar el funcionamiento del sistema.

## Objetivo

El objetivo de este proyecto es modelar una base de datos basada en el analisis de una plataforma de empleo, tomando como referencia la pagina El Empleo, para transformar sus caracteristicas y requisitos observados en una solucion estructurada en SQL.
